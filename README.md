# JoMa Sessions - UX/UI Design Concept

## Project Overview
High-fidelity prototype for a music session platform designed to connect artists with live session recordings. The focus was on a **mobile-first approach**, accessible navigation, and a dark-mode aesthetic consistent with the brand's identity.

### Demo Walkthrough
https://github.com/user-attachments/assets/663b140f-c80e-437e-bbe7-693553117af7

## Key Features
* **Interactive Player:** Custom controls designed for ease of use on mobile screens.
* **Visual Hierarchy:** Prioritized content discovery with clear typography and spacing.
* **Prototyping:** Full user flow connected in Figma (Login -> Home -> Player).
